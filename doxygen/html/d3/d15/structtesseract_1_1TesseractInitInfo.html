Detailed Description

Used to store information about how a given Tesseract was initialized. See the Tesseract init methods.

Note: Recreating a Tesseract does not guarantee that it is identical to the Tesseract associate with this construction info since the kinmeatics managers could have changed and it does not include the environment command history. This will simply recreate the Tesseract as it was at construction.
