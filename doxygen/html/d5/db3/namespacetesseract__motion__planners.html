Make a vector of position samplers from a vector of waypoints for a robot attached to a kinematic positioner.

This is to be used if you have a robot attached to a rail, gantry, revolute positioner, mobile base, ext. where the forward kinematics object represent the kinematics of the object the robot is attached to. The tip of the forward kinematics object must match the base link of the inverse kinematics object for the robot.
