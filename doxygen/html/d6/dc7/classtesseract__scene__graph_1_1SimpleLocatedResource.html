tesseract_common::Resource::Ptr tesseract_scene_graph::SimpleLocatedResource::locateSubResource ( const std::string &  relative_path)
overridevirtual

Retrive a sub-resource relative to the current resource.

Retrive a sub-resource related to the current resource. For instance, retrieve an image resource for a mesh. The relative_path should be relative to the parent directory containing the current resource.

This function is optional, and will only be available on resource locators that support retrieving relative resources.

Parameters
relative_pathPath relative to the parent directory of the current resource
Returns
Resource::Ptr The located resource, or nullptr if not found
