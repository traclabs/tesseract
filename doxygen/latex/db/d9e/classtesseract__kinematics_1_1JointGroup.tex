\doxysubsection{Detailed Description}
A Joint Group is defined by a list of joint\+\_\+names. 

Provides the ability to calculate forward kinematics and jacobian. \begin{DoxyNote}{Note}
This creates an optimized object replace all joints not listed in the provided list with a fixed joint calculated using the provided state. Also the calc\+Fwd\+Kin only return the link transforms in the optimized object which is all active links and root of the scene graph usually. 
\end{DoxyNote}
