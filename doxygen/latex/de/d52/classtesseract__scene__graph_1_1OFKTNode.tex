\doxysubsection{Detailed Description}
The OFKT node is contains multiple trasformation which are described below. 


\begin{DoxyItemize}
\item Static Transformation\+: (S)
\item \mbox{\hyperlink{classtesseract__scene__graph_1_1Joint}{Joint}} Transformation\+: (J) this if a fuction of the joint value if not fixed
\item Local Transformation\+: L = S $\ast$ J
\item World Transformation\+: W = W(parent) $\ast$ L 
\end{DoxyItemize}
