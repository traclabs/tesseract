Add a collision object to the checker. 

All objects are added should initially be added as static objects. Use the set\+Contact\+Request method of defining which collision objects are moving.


\begin{DoxyParams}{Parameters}
{\em name} & The name of the object, must be unique. \\
\hline
{\em mask\+\_\+id} & User defined id which gets stored in the results structure. \\
\hline
{\em shapes} & A vector of shapes that make up the collision object. \\
\hline
{\em shape\+\_\+poses} & A vector of poses for each shape, must be same length as shapes \\
\hline
{\em shape\+\_\+types} & A vector of shape types for encode the collision object. If the vector is of length 1 it is used for all shapes. \\
\hline
{\em collision\+\_\+object\+\_\+types} & A int identifying a conversion mode for the object. (ex. convert meshes to convex\+\_\+hulls) \\
\hline
{\em enabled} & Indicate if the object is enabled for collision checking. \\

\end{DoxyParams}
