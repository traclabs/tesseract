Retrive a sub-\/resource relative to the current resource. 

Retrive a sub-\/resource related to the current resource. For instance, retrieve an image resource for a mesh. The relative\+\_\+path should be relative to the parent directory containing the current resource.

This function is optional, and will only be available on resource locators that support retrieving relative resources.
