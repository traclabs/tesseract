Calculates joint solutions given a pose for each tip link. 

This interface supports IK for both kinematic chains that have a single tool tip link and kinematic chains that have multiple tip links. For example, consider a robot with external part positioner\+: a pose can be specified to be relative to the tip link of the robot or the tip link of the positioner is to support a pose relative to a active link. For example a robot with an external positioner where the pose is relative to the tip link of the positioner. \begin{DoxyNote}{Note}
Redundant joint solutions can be provided by the utility function get\+Redundant\+Solutions 
\end{DoxyNote}
